Connect GitHub API

Add a fine-grained token to unlock features that need authenticated GitHub requests.

To create a fine-grained GitHub token, visit GitHub Settings > Personal access tokens > Fine-grained tokens. Select "All repositories", and grant read-only access to "Metadata" and "Contents" permissions.
